
# React.js TicTacTo online game 🎮


## Features 🔥

- private rooms 🔒
- realtime ⚡️
- join using ID & LINK 🔌
- create new rooms 🎉


## Tech Stack

**Client:** React.js

**Server:** Express.js


## Installation & setup 🛠

You should have node installed in your device

Clone the project

```bash
  git clone https://github.com/AhmedSama/react.js-tictacto-online.git
```

Go to the project directory

```bash
  cd react.js-tictacto-online
```

go to the server directory and run

```bash
  npm install
```

Start the server

```bash
  npm run start
```
go back to the project directory and cd to tictacto directory

```bash
  cd ..
  cd tictacto
```
in tictacto directory run
```bash
  npm install
```
after that run
```bash
  npm run start
```
